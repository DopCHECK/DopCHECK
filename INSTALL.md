# Installation

## Prerequisites

* Python Version 3.8 or later

## Python Library Dependencies
* os
* time
* re
* spacy (model en_core_web_sm is required, see spaCy website here: https://spacy.io/models/en) or use ''python -m spacy download en_core_web_sm'' 
* BeautifulSoup
* csv
* urllib
* json
